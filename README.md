# merge_game
An AI generated game

## Sound Effects and Music Attribution

All sound effects and background music in this game are generated programmatically using the Web Audio API. These are original works created specifically for this game and are in the **public domain**.

### Audio Features
- **Drop Sound**: A descending tone played when fruits are released
- **Merge Sound**: A cheerful ascending chime played when fruits merge
- **Background Music**: A continuous looping melody during gameplay that fades out when the game is paused or ended

All audio is synthesized in real-time using Web Audio API oscillators and is therefore free from any copyright restrictions.
